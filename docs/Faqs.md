# Pregunta 1

¿Qué debo evaluar antes de contratar un servicio de portería o vigilancia?

Solidez tecnológica: Foxsys utiliza el premiado software de videovigilancia Milestone Systems y un sistema de información y gestión propio para brindar servicios de portería en edificios.

Seguridad real: Homologada por el Ministerio del Interior, Foxsys cuenta con conexión directa al centro de comando unificado para despachar móviles policiales en segundos. 

Trayectoria y servicio: Con decenas de miles de usuarios mensuales, Foxsys cuenta con un equipo especializado, respaldado por un NPS (índice de recomendación) superior al 60%.

# Pregunta 3

¿Cómo funciona la seguridad?

Foxsys previene eventos de seguridad mediante una protección perimetral con cámaras que cuentan con analíticas de inteligencia artificial. Ante una actividad sospechosa, un operador del centro de monitoreo disuade utilizando el altoparlante. Si persiste la amenaza, se envía un móvil policial de inmediato. Además, contamos con personal y vehículos propios para abordar cualquier eventualidad.
